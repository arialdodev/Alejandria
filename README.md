## Este proyecto lo hice para reforzar los siguientes temas.

Alejandria es una pagina web de biblioteca digital desarrollada en la tecnología C# que utiliza el framework Entity Framework Core para la gestión eficiente de bases de datos. El proyecto sigue una arquitectura de capas bien definida y emplea el patrón de diseño Builder para facilitar la construcción y manipulación de objetos complejos. Además, utiliza Bootstrap para mejorar el diseño y la presentación de algunas tablas en la interfaz de usuario.
